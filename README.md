Projekt: Task manager

Program umoznuje přidávat, zobrazovat a odstraňovat úkoly do seznam ukolu. 

Technologie:
Python 3.11.2

Instalace:
1. Naklonujte repozitář:
   ```bash
   git clone https://github.com/Elibon123/Project1.git
   ```
2. Přejděte do složky projektu:
   ```bash
   cd manager.py
   ```

Pouziti

Program se spouští přímo z terminálu. Po spuštění se zobrazí textové menu, ve kterém se pohybujete zadáváním čísel `1` až `4`.

1. Spuštění programu

Spusťte skript ve svém terminálu:

bash

    py manager.py

2. Ukázka práce s programem

Po spuštění uvidíte hlavní menu:

    Spravce ukolu - Hlavni menu
    1. Pridat novy ukol
    2. Zobrazit vsechny ukoly
    3. Odstranit ukol
    4. Konec programu
    Vyberte moznost (1-4): 1
    

Přidání úkolu (Volba 1):

    Zadejte nazev ukolu: Nakoupit
    Zadejte popis ukolu: Rohliky, mleko a maslo
    Ukol 'Nakoupit' byl pridan!


Zobrazení úkolů (Volba 2):

    Seznam ukolu:
    1. Nakoupit - Rohliky, mleko a maslo
    

Odstranění úkolu (Volba 3):

    Zadejte cislo ukolu, ktery chcete odstranit: 1
    Ukol 'Nakoupit' byl odstraněn!
    

Ukončení programu (Volba 4):

text:

    Konec programu.
